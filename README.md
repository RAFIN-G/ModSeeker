"ModSeeker" is a lightweight Paper plugin for Minecraft servers.  
It works together with the companion Fabric mod "Hidder" to let server admins check which mods a player is running~ 

### Features
- Detects client-side mods from players using the Hidder mod.  
- Helps admins enforce server rules against unwanted mods.  
- Lightweight, fast, and easy to configure.  

### Requirements
- Fabric client with the Hidder mod installed
